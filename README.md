# MATECH - Landing Page<br>

## Sitio web responsive desarrollado para MATECH, una empresa orientada a soluciones informáticas y soporte técnico.<br><br>

### Tecnologías utilizadas<br>
  • **HTML5**<br>
  • **CSS3**<br>
  • **JavaScript**<br>
  • **AOS (Animate On Scroll)**<br>
  • **EmailJS**<br>
  • **Font Awesome**<br><br>

### Características<br>
  • Diseño moderno y responsive<br>
  • Menú hamburguesa para dispositivos móviles<br>
  • Animaciones con AOS<br>
  • Formulario de contacto funcional mediante EmailJS<br>
  • Botón de contacto rápido por WhatsApp<br>
  • Secciones informativas sobre servicios<br>
  • Landing page optimizada para deploy en Vercel<br><br>

El sitio se adapta correctamente a:<br>

  • Tablets<br>
  • Dispositivos móviles<br><br>

### Configuración de EmailJS<br>
Para utilizar el formulario de contacto:<br>
  • Crear una cuenta en EmailJS.<br>
  • Configurar un servicio de correo.<br>
  • Crear una plantilla.<br>
  • Reemplazar los siguientes datos:<br><br>

*emailjs.init("TU_PUBLIC_KEY")*<br>
*emailjs.sendForm(*<br>
    *"TU_SERVICE_ID",*<br>
    *"TU_TEMPLATE_ID",*<br>
    *this*<br>
*)*<br><br>
